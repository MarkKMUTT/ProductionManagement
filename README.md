---

ฟอร์มกรอกข้อมูลการผลิตประกอบด้วย
- **วันทีผลิต**
- **ไลน์การผลิด**
- **รุ่นที่ผลิต**
- **จำนวนที่ผลิต**
ปุ่ม Input ส่งฟอร์มไปที่ตารางเมื่อใส่ข้อมูลครบถ้วน และปุ่ม Clear Form สำหรับลบฟอร์ม

Table Report
สรุปผลจำนวนรายการและจำนวนที่ผลิตทั้งหมดที่ได้รับมาจากฟอร์ม และปุ่ม Delete
